## 介绍

`Anemo` 一词来自希腊语中的 `风` 一词，象征着框架的轻量与诗意，也饱含开发者的热忱，框架使用[TypeScript](https://www.typescriptlang.org/)

详情参考使用文档: https://anemo-docs.pages.dev/
